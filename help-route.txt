Routes: 

/ : returns a welcoming message 

/help: returns description of each route

/incidents: DEALS with data depending on the method called -- change the word deals

   Possible Methods: 

     1. POST: initializes data 
     2. GET: returns data 
     3. DELETE: deletes data 

/incident: returns list of all incidents of a given incident type

/epochs: returns list of epochs

/ids: returns list of incident identification numbers

/issues: returns list of incident types

/published-range: returns earliest and latest published incident dates

/coordinates-range: returns dictionary of maximum and minimum coordinates
